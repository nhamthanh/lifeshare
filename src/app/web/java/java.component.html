Depoly backend API with spring boot on heroku

September 13th 2019
Spring boot is a collection of spring frameworks help you to create and build a web app. In this section, we use spring boot as backend api which allow us to seperate service and interface or implement microservices. Heroku is free host for us to deploy projects with limited, you will need to pay fee to using full services. Pros: Used for medium or large project. Auto deploy when submit code to master on github. Cons: Cost for backend and server.
Environment for spring boot API
  1. Download and Install jdk at https://www.oracle.com/technetwork/java/javase/downloads/jdk12-downloads-5295953.html: Environment for java developer
  2. Download and Install maven at https://maven.apache.org/download.cgi
  3. Setting JDK_HOME and MAVEN_HOME
  4. Download eclipse IDE at https://www.eclipse.org/downloads/: IDE used for java
  5. Open eclipse, create new maven project
  6. Open pom.xml and declaring spring boot dependencies <pre> <dependency> <groupId>org.springframework.boot</groupId> <artifactId>spring-boot-starter-web</artifactId> </dependency> <dependency> <groupId>org.springframework.boot</groupId> <artifactId>spring-boot-starter-test</artifactId> <scope>test</scope> </dependency> </pre>
  7. Right click on project, choose run as -> maven build (input clean install for Goals)
  8. At src folder, create new application and controller file. (Package must follow convention, so that spring will auto scan file)
Deploy using Heroku Git
  1. Register account in heroku page
  2. Click on New to create new heroku app
  3. Download and install the Heroku CLI
  4. run 
    $ heroku login
  5. Clone the repository. Use Git to clone thanhncspring's source code to your local machine.
  6. run 
              $ heroku git:clone -a <your_project>
          $ cd <your_project>
  7. Deploy your changes. Make some changes to the code you just cloned and deploy them to Heroku using Git.
  8.           $ git add .
          $ git commit -am "<your_comment>"
          $ git push heroku master
  9. Choose your app
  10. Click on Open app
