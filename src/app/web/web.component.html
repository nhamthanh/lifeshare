Create a free website with angular and github pages

September 6th 2019
You can host a free website, blog create by your self with github pages and angular framework. Angular framework make your code more easier to maintain, support you with fontend library, third party library, theme ... If you don't want to use angular, just create a simple html website. Github is your repository where you can store your code and deploy your website with github pages. In this content, I will show you how to create a static web with angular and deploy your website.
Environment
  1. Register an account in github https://github.com/
  2. Install Node.js and npm in https://nodejs.org/en/download/. This is environment to run and build angular framework
  3. Install angular-cli for npm https://cli.angular.io/. This is collection of angular command
  4. Download an IDE to coding Ex: https://code.visualstudio.com/download
Create angular project 
  1. ng new < project name >
    in cmd to create your project
  2. ng server
    in cmd and test your website in localhost page http://localhost:4200/. It is default port for angular project
Connect to github
  1. In https://github.com/, create your repository, it is usually same with project name.
  2. Go to code tab and follow the step â€¦or push an existing repository from the command line
  3. Run cmd in your porject folder and run the command. Then your code have been pushed to github respository.
Deploy your free website with githubs pages
  1. install angular-cli-ghpages to help you do that in https://alligator.io/angular/deploying-angular-app-github-pages/
  2. ng build --prod --base-href "https://<user-name>.github.io/<repo>/"
    This will build angular project into html, javascript in dist folder (you should move content inside project folder in dist to outside)
  3. ngh
  4. open website https://<user-name>.github.io/<repo>/ your can see your website.
